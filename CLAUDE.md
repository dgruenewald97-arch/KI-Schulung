# Projektkontext

Dies ist jetzt eine bewusst einfache statische Website für GitHub Pages.

Keine React-App, kein Vite, kein Build-Schritt.

## Dateien

- `index.html` enthält die sichtbare Struktur.
- `assets/styles.css` enthält das Styling.
- `assets/app.js` enthält Inhalte, Rollen, Kundentypen, Prompt-Vorlagen, Quiz und Werkstatt-Logik.
- `.github/workflows/deploy.yml` lädt die statischen Dateien direkt zu GitHub Pages hoch.

## Ziel

Die Schulung soll Mobility-Minds-Mitarbeitenden helfen, kundenzentrierte Prompts für Langdock zu bauen.
Wichtig sind Agenturkontext, Kundentyp, Zielgruppe, Kanal, Material, Tonalität, Datenschutz und Ergebnisprüfung.
